# money-transfer-ws

Simple money transfer web service.

## Functionality

-   simple account management
-   money transfers between accounts

## API & Limitations

-   REST API
-   authentication unsupported
-   in-memory storage

## Technologies

-   Java 8
-   Jersey Web Services / JAX-RS
-   Jackson JSON
-   Maven

## Build & Run

-   build: `mvn clean package`
